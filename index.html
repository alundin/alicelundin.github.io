Coll

Coll was developed during the hackathon East Sweden Hack 2014 in Linköping, Sweden. It is an application that keeps track of your movements during the day by accessing the GPS in the phone. It categorises the movements and visualizes the users own carbon dioxide emissions, lets the user set goals to improve his/her carbon footprint and follows up on the development. Coll also suggests public transfers by the open data of Östgötatrafiken. Länk till github

Incognito

Incognito was developed durin the 24 hour hackathon Hack for Sweden 2015 in Stockholm, Sweden. The website is a recruitment portal with a main vision to be neutral and non-discriminating. Open data from Arbetsförmedlingen is used to get job offers to start with, this makes the website useful from scratch. Applicants can sign up to be able to apply for jobs without attaching a photo or given name, but only competence. Recruiters, who possibly have a problem of always hiring "the same person" can contact applicants without knowing gender, age or etnicity - only competence!

Incognito aims to be an eye-opener to recruiters and a new, non-discriminating way to apply for jobs!

Purgastory

Purgastory is an adventour game created in Unity3D during a project course about 3D graphics. The game starts off with the player dying and ending up between heaven and hell - in Purgatory. The player is then challanged in different ways, in two levels, where to end up. There is no "win" or "loose", it all depends on where the player wants to end up!
